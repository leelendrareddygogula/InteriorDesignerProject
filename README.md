<h2>Interior Designer Platform</h2>
<p>This repository contains the backend implementation for a professional Interior Designer website. Built with Spring Boot, the system provides a structured and scalable architecture to manage design portfolios, client inquiries, and service listings.</p>
<h3>Features</h3>
<ul>
  <li><b>Portfolio Management: </b>RESTful endpoints to manage high-resolution project images, categories, and descriptions, allowing for a dynamic showcase of design work.</li>
  <li><b>Lead Generation & Inquiries: </b>Secure handling of client contact forms and design consultation requests.</li>
  <li><b>Service Categorization: </b>Organized data structures to categorize services such as Residential, Commercial, and Landscape design</li>
  <li><b>Modular Backend Design: </b>Decoupled architecture using Spring Boot to ensure the system can easily integrate with various frontend frameworks or third-party CRM tools.</li>
  <li><b>Data Persistence: </b>Optimized database interactions for storing project metadata and client interactions.</li>
</ul>
<h3>Tech Stack</h3>
<ul>
  <li><b>Language: </b>Java</li>
  <li><b>Framework: </b>Spring Boot</li>
  <li><b>Web: </b>Spring MVC (REST APIs)</li>
  <li><b>Frontend Components: </b>HTML, CSS</li>
  <li><b>Build Tool: </b>Maven</li>
  <li><b>Database: </b>MySQL</li>
</ul>
<h3>Implementation Overview</h3>
<h4>Architecture</h4>
<p>The project follows the standard Model-View-Controller (MVC) pattern</p>
<ul>
  <li><b>Controllers: </b>Handle incoming HTTP requests for portfolio and inquiry data.</li>
  <li><b>Services: </b>Contain the business logic for processing design project updates and form submissions.</li>
  <li><b>Repositories: </b>Manage the abstraction layer for database operations.</li>
</ul>
<h4>Styling & Presentation</h4>
<p>While primarily a backend project, it includes integrated CSS and HTML components to ensure a cohesive look and feel for any server-side rendered pages or administrative dashboards.</p>
